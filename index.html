My AI Journey: 비개발자에서 시스템 개발자로

1. 시작: "왜 아무도 이 불편함을 해결해주지 않을까?"

반복되는 수작업, 예측 불가능한 리스크, 그리고 휴먼 에러로 인한 손실. 현장의 비효율을 해결하고 싶었지만, 기존의 툴과 프로세스는 명확한 한계가 있었습니다. 개발팀에 의뢰하기엔 너무 사소하고, 기다리기엔 너무 시급한 문제들 앞에서, '내가 직접 만들 수 있다면'이라는 생각을 품게 되었습니다.

2. 과정: AI와의 협업을 통한 성장

코딩의 'ㅋ'도 몰랐지만, 저에겐 강력한 동료가 있었습니다. 다양한 AI 플랫폼을 'AI 스승'이자 '24시간 페어 프로그래머'로 삼아 끊임없이 질문하고 학습했습니다.

  • Gemini, GPT, Claude: 기본 문법부터 복잡한 업무 로직을 자연어로 설명하고 최적 코드로 변환
  • Windsurf, Grok: 실시간 코드 리뷰 및 개선 제안을 통한 코드 품질 향상
  • Kimi, DeepSeek: 대용량 데이터 처리 로직 최적화 및 알고리즘 개선
  • Perplexity: 최신 API 활용법 및 기술 트렌드 리서치로 지속적인 역량 강화

3. 결과: 상상을 현실로 만드는 힘

수백, 수천 번의 질문과 테스트 끝에, 마침내 Google Apps Script(GAS)를 기반으로 한 완전한 기능의 웹앱을 제 손으로 완성할 수 있었습니다. 단순 반복 업무의 패턴을 AI 학습으로 발견하고, 이를 자동화 시스템으로 구현하여 데이터 입력 즉시 정보가 완성되는 실시간 처리 환경을 구축했습니다.

일일 원가 및 생산성 집계 시스템

MES-ERP 연계를 통해 일일 제조원가를 실시간으로 모니터링하고 표준원가와의 차이를 수율/생산성 차이로 자동 분석하여 리포팅합니다.

주요 기능:

  • BCG 매트릭스 모델링 기반 데이터 세그먼트
  • 이상 징후 발생 시 담당자 자동 알림 (Google Mail)
  • 경영진 원클릭 실적 확인 및 피드백 메일 기능

실시간 재고 및 소비기한 관리

재고 현황과 소비기한을 실시간으로 관리하여 위험도를 선제적으로 알리고, 장기재고는 별도 세그먼트로 분류하여 손실을 최소화합니다.

주요 기능:

  • 소비기한 스트레스 테스트 (0~365일 조절)
  • 판매 가능/불가능 재고 즉시 파악
  • 장기/위험 재고 자동 알림 서비스

AI 기반 근무시간 예측 시스템

근태 데이터를 기반으로 개인별 잔여 근무시간을 일 단위로 예측하여 52시간제 위반 리스크를 원천 차단하고 인력 운영을 최적화합니다.

주요 기능:

  • 근무시간 예측 시뮬레이션
  • 구간별(경고/위험) 자동 알림
  • 주말/연장근로 가용 인원 즉시 조회

실시간 생산성 분석 대시보드

인당/인시당 생산성을 실시간으로 추적하여 목표 대비 실적을 관리하고, 인원 증감 현황과 생산성 트렌드를 즉시 파악합니다.

주요 기능:

  • 목표 대비 실적 Gap 자동 분석
  • 인원 변동 현황 실시간 트래킹
  • 생산성 저하 요인 드릴다운 분석

일일 실적 마감 모니터링 시스템

일일 마감 실적 처리 현황의 진척률을 실시간으로 모니터링하고, 미비한 부분은 담당자에게 '핀셋 알림'을 보내 누락을 방지합니다.

주요 기능:

  • 마감 진척률 실시간 대시보드
  • 미진 내용(오더명 등) 자동 식별
  • 담당자별 핀셋 알림 서비스

재료비 원단위 심층 분석 시스템

재료비 변동 요인을 '내부 요인(수율/효율)'과 '외부 요인(구매단가)'으로 명확하게 분리하여 분석하고, 원가 개선 포인트를 도출합니다.

주요 기능:

  • 원단위 차이 자동 분석 (수율/효율)
  • 자재별 구매 단가 트렌드 추적
  • 원가 절감 기회 요인 자동 리포팅

제조비용 실적 모니터링 시스템

ERP 전표와 연계하여 일간/월간 제조비용 실적을 예산 및 전년/전월과 실시간으로 비교 분석하고, 이상치는 자동으로 알림을 보냅니다.

주요 기능:

  • 예산 대비 집행 현황 실시간 추적
  • 다차원(전년/전월) 트렌드 분석
  • 이상 비용 발생 시 자동 알림

자재 구매 최적화 시스템

과거 2년간의 자재 사용량 및 구매 단가 데이터를 심층 분석하여 변동성을 예측하고, 구매 전략에 대한 데이터 기반의 액션 플랜을 제안합니다.

주요 기능:

  • 관리 위험도 자동 세그먼트화
  • 데이터 기반 구매 액션 카드 자동 제안
  • 공급망 리스크 사전 경고 시스템

S&OP 연계 대시보드

수주, 생산, 재고 데이터를 통합하여 생산 진척률을 실시간으로 추적하고, 고객 납기 대응 능력을 극대화하여 신뢰도를 높입니다.

주요 기능:

  • 납기 지연 가능성 사전 예측 및 경고
  • 재고 연계 생산 계획 자동 조정 제안
  • 고객사별 주문 이행 현황 실시간 추적
